# Análise-Bolsa

* **templates** - *Documentos HTML*
  - layout.html
  - index.html
* **static** - *CSS e JavaScript*
  * **CSS** - *CSS para estilo da página*
    - styles.css
  * **JS** - *Scripts JS processados pelo navegador*
    - app.js
* **pycache** - *Cache*
- app.py - *Aplicativo principal python+flask*

**Módulos utilizados:**
  * *Flask*
    * *Flask-Bootstrap*
  * *YFinance*
  * *HTML5lib*
  * *BeautifulSoup4*
